# Valentine Page

## Customize the text
Update the names and message in `script.js`:

- `herName`
- `yourName`
- `revealMessage`

## Add your photos
Drop your photos into `assets/` and name them:

- `photo-1.jpg`
- `photo-2.jpg`
- `photo-3.jpg`

Or change the `src` in `index.html` to point at your own filenames.

## Swap the GIFs
Edit the two `img` tags in the GIF section inside `index.html`.

## Deploy on GitHub Pages
1. Create a new repo and put these files in the repo root.
2. Commit and push.
3. In GitHub: `Settings` -> `Pages` -> select branch `main` and `/ (root)`.
4. Your site will be available at:

```
https://<username>.github.io/<repo>/
```

If you use the special repo name `<username>.github.io`, the site will be:

```
https://<username>.github.io/
```
